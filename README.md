# PokéDex Interativa

Projeto desenvolvido para a disciplina de Web Development Front-End, com o objetivo de aplicar conceitos de consumo de API, manipulação de DOM e responsividade utilizando tecnologias web fundamentais.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PokéAPI

## Funcionalidades

- Consumo de dados em tempo real via API pública
- Renderização dinâmica de cards Pokémon
- Busca por nome
- Busca por número da Pokédex
- Filtro por tipo
- Modal interativo com:
  - Imagem oficial
  - Estatísticas base
  - Tipos
  - Fraquezas
  - Descrição
- Dashboard visual com barras de status
- Layout responsivo

## Objetivos Acadêmicos

Este projeto foi desenvolvido para demonstrar:

- Integração com APIs
- Uso de `fetch()`
- Manipulação dinâmica do DOM
- Organização estrutural com HTML semântico
- Estilização moderna com CSS
- Interatividade com JavaScript puro
